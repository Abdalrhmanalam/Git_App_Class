# مشروع كلاسات جافا | Java Classes Project

## وصف المشروع | Description
هذا المشروع عبارة عن تطبيق **جافا** (Java) يوضح كيفية إنشاء الكلاسات (Classes) واستخدامها، مع تعريف الخصائص (Attributes) والدوال (Methods). يهدف المشروع إلى تطبيق مفاهيم البرمجة الكائنية (OOP) مثل التغليف (Encapsulation) والوراثة (Inheritance).

This is a **Java** project that demonstrates how to create and use classes, with defined attributes and methods. The project aims to apply Object-Oriented Programming (OOP) concepts such as Encapsulation and Inheritance.

## الميزات الرئيسية | Key Features
- إنشاء كلاسات متعددة | Creating multiple classes
- تعريف الخصائص (Attributes) مع معدلات الوصول المناسبة | Defining attributes with proper access modifiers
- استخدام الدوال (Methods) للتفاعل مع الخصائص | Using methods to interact with attributes
- تطبيق مفهوم التغليف (Encapsulation) | Implementing encapsulation
- توثيق الكود (Code Documentation) | Code documentation

## التقنيات المستخدمة | Technologies Used
- **اللغة**: Java 
- **الإصدار**: Java 8 أو أحدث | Java 8 or later
- **الأدوات**: أي IDE يدعم Java (IntelliJ IDEA, Eclipse, VS Code)
